Welcome to the **Grocery Store website**, a fully interactive online grocery demo.  
Users can browse products, search for items, add them to a shopping cart, and log in to access personalized features.

---

## **Website Features and User Flow**

1. **Homepage (`index.html`)**  
   - Users land on a clean and visually appealing homepage.  
   - Displays featured products and banners for promotions.

2. **Search Functionality**  
   - Users can **search for any food item** using the search bar.  
   - Search results display matching products immediately, making it easy to find what they need.

3. **Product Browsing**  
   - Scroll through all available grocery items with images and descriptions.  
   - Hover effects enhance the user experience.

4. **Add to Cart**  
   - Users can **add items to their shopping cart** with a single click.  
   - The cart updates dynamically to show the total number of items and prices.

5. **Login Functionality**  
   - Users can log in to access personalized sections.  
   - Login form validates input and simulates user authentication.

6. **Blog Page (`blog.html`)**  
   - Contains articles about groceries, tips, and recipes.  
   - Supports images for a visually rich experience.

7. **Responsive Design**  
   - Website is fully responsive using CSS, providing a smooth experience on **desktop, tablet, and mobile**.  

8. **Interactive JavaScript**  
   - Smooth scrolling, dynamic cart updates, and hover effects enhance usability.
